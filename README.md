# flashlight-icli
![flashlight-icli](https://github.com/WayneKim92/flashlight-icli/assets/75321423/8b8c1d55-0cef-4351-84e3-086198c22c8f)

Interactive Command Line Interface for flashlight

## Commands
```
npx flashlight-icli init
npx flashlight-icli
```
